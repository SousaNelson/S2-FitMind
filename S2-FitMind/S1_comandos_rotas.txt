Para os pacientes:
/pacientes --> GET patients: para pegar todos os pacientes.
/patients/id --> GET patients/id: pegar um patients com base no seu id.
/patients --> POST: criar um novo cliente.
/patients --> DELETE: patients por id para deletar um patient especifico
/patients --> PUT: atualiza um patient já criado.
